    Wallure

    Task: Completely redesign an e-commerce store for a wall paneling manufacturer.

    For this project I partnered with a great developer Ivetka Tonhauserova and we designed and developed a whole new store for the company.

    Deliverables:

    Landing Page & App

    Honeycomb

    Task: To design a landing page and a web app for a tax return service. To make tax returns fun.

    I loved designing this project as I got a lot of freedom from the founders. This is simply one of those feel-good projects that you just enjoy working on.

    Deliverables:
    • Design direction
    • Landing page in Figma
    • Style guide
    • Desktop app UI/UX in Figma
    Challenges:

    When you get a complete freedom in design direction, sometimes you end up just... going in the wrong direction. First draft of the landing page was "too cute" for the client. I realised we could use a style guide to show just enough of elements to confirm the design direction. After I created that and client felt it's right, it was a breeze creating all the screens.
